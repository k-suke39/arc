## サービス URL

https://current-user.vercel.app/

## 概要

current_user は環境構築不要、Ruby on Rails の O/R マッパーである ActiveRecord の学習サービスです。

## 主な機能

- ログイン機能
- ログアウト機能
- 問題選択機能
- ActiveRecord 実行機能
- ActiveRecord 実行時にタブ切り替え機能
- ActiveRecord 判定機能
